# Synaplan Next — Projektübersicht

## Vision

Neuaufbau der **synaplan.com** als moderne, performante **Next.js 16.2**-Website für die Synaplan Open-Source AI-Management-Plattform. Die Seite soll Sales-orientiert sein, das Produkt (Chat Widget, Memories, Multi-Model-Routing) in den Vordergrund stellen und durch eine professionelle Informationsarchitektur Vertrauen aufbauen. Durch die AI/Agent-Features von Next.js 16.2 (AGENTS.md, Agent DevTools, Browser Log Forwarding) ist die Seite optimal für Vibe-Coding-Erweiterung aufgebaut.
